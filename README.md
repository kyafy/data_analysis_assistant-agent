# 智能数据分析助手 (Data Analysis Assistant)

基于大模型（LLM）的智能数据分析 Agent，支持通过自然语言查询数据库、自动生成 SQL、执行查询并可视化展示结果。

## ✨ 主要功能

- **自然语言交互**：直接使用自然语言提问，无需编写 SQL。
- **智能 SQL 生成**：基于 LangChain 和 Qwen 模型，自动将自然语言转换为 SQL 语句。
- **自动可视化**：根据查询结果自动推荐并生成 ECharts 图表（柱状图、折线图、饼图等），支持混合展示数据表格。
- **流式响应**：支持 SSE (Server-Sent Events) 流式输出，实时展示思考过程、SQL 生成与执行结果。
- **动态数据库配置**：前端提供可视化配置界面，支持运行时热切换不同的 MySQL 数据库连接。
- **会话管理**：支持多会话管理，保存历史对话上下文。

## 🛠️ 技术栈

### Backend (后端)
- **Framework**: FastAPI
- **LLM Orchestration**: LangChain
- **Database ORM**: SQLAlchemy
- **Model**: Qwen (通义千问) via DashScope/OpenAI API compatible
- **Tools**: Pandas, PyMySQL

### Frontend (前端)
- **Framework**: React + Vite
- **UI Components**: TailwindCSS + Lucide React
- **State Management**: Zustand
- **Visualization**: Apache ECharts (echarts-for-react)

## 🚀 快速开始

### 1. 环境准备
- Python 3.10+
- Node.js 18+
- MySQL 5.7/8.0 (推荐使用 Docker 启动)

### 2. 启动数据库
项目中提供了 `docker-compose.yml` 快速启动 MySQL 环境:
```bash
docker-compose up -d
```
启动后，导入测试数据：
```bash
mysql -h 127.0.0.1 -P 3307 -u root -proot data_analysis < init_mock_data.sql
# 导入复杂测试数据 (包含时间序列和多维分析数据)
mysql -h 127.0.0.1 -P 3307 -u root -proot data_analysis < insert_complex_mock_data.sql
```

### 3. 后端启动
```bash
cd backend

# 推荐使用 uv 管理依赖，或者使用 pip
# 安装依赖
pip install -r requirements.txt  # 需自行导出，或参考 pyproject.toml

# 配置环境变量
# 在 backend 目录下创建 .env 文件，填入 LLM API KEY
# DASHSCOPE_API_KEY=sk-xxxxxxxx

# 启动服务
python main.py
```
后端服务默认运行在 `http://localhost:8000`。

### 4. 前端启动
```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
前端服务默认运行在 `http://localhost:5173`。

## 📝 使用说明

1.  **配置数据库**：点击左下角的设置图标（⚙️），输入数据库连接信息（默认 Host: 127.0.0.1, Port: 3307, User: root, Password: root, DB: data_analysis）。
2.  **开始对话**：点击“新建对话”，在输入框中输入你的数据分析需求。
    *   *示例 1*: "帮我统计每个部门的总销售额"
    *   *示例 2*: "分析 2023 年第一季度电子产品的销量趋势，用折线图展示"
    *   *示例 3*: "列出销售额最高的前 3 名员工"
3.  **查看结果**：右侧可视化面板将自动展示查询到的**数据表格**，如果适用，还会展示自动生成的 **ECharts 图表**。
